# Generate Parentheses

https://leetcode.com/problems/generate-parentheses/description/

## Problem Description

Given `n` pairs of parentheses, write a function to generate all combinations of well-formed parentheses.

## Example 1:

```text
Input: n = 3
Output: ["((()))","(()())","(())()","()(())","()()()"]
```

## Example 2:

```text
Input: n = 1
Output: ["()"]
```

## Constraints:

```text
1 <= n <= 8
```
